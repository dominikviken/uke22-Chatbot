Laget chatbot med enkle svar.
Kjør main.py for å bruke chatbotten. Du kan legge til svar inn i Python skriptet i "svar" ordboken.

Du kan lage egen Chatbot ved å følge OpenAI dokumentasjon. De bruker JSON filer til å trene chatbotter, der de gir de "prompts" og svar til promptene, som AI-en da følger.
